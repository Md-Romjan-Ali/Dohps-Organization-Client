# 🌱 DOHPS Organization

A modern, responsive NGO website developed for **Development Organization for Human Potential Sustainability (DOHPS)**. The platform showcases the organization's mission, programs, scholarship initiatives, success stories, donation system, and community impact while providing an easy way for visitors to connect and contribute.

## 🌐 Live Website

**Live Site:** https://dohps-organization-client-rzsh-2wcyez34v.vercel.app/

## 📖 About DOHPS

Development Organization for Human Potential Sustainability (DOHPS) is a non-profit organization established in 2008 with the goal of promoting sustainable human development through education, health, social empowerment, and community development. The organization primarily supports talented students from financially disadvantaged families and works to improve the quality of life in vulnerable communities across Bangladesh.

# ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Go to the project folder

```bash
cd project-name
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Start production server

```bash
npm start
```

---

# 📁 Environment Variables

Create a `.env.local` file and add your environment variables.

```env
NEXT_PUBLIC_API_URL=your_api_url

MONGODB_URI=your_mongodb_uri

BETTER_AUTH_SECRET=your_secret

BETTER_AUTH_URL=http://localhost:3000
```
